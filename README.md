<h2>рҹ“Ң Project Overview</h2>

<p>
The <b>Employee Management System</b> is a full-stack web application that allows organizations to manage employees and departments efficiently. 
The system provides features to perform CRUD operations on employees and departments while maintaining the relationship between them.
</p>

<p>
This project demonstrates modern full-stack development using <b>Spring Boot for backend APIs</b> and <b>React for frontend UI</b>.
</p>
